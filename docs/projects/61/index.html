About this project:

Our Project addresses the need for a Turntable for Dj‚Äôs with limited hand dexterity, like our volunteer Alex. It‚Äôs very difficult for Alex to use a normal DJ table so we came up with a single turntable and a pedal with all the components he needs in order to be able to use his DJ setup with one hand.

The pedal Would help with changing the music tracks. Neutral position for track 1, halfway press to merge the tracks and full press down for track 2. The turntable would be formatted so that it can be operated with one hand.
